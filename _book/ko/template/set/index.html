set

지역변수처럼 변수에 값을 할당해 사용할 수 있습니다.

    {% set version = "1.0.0" %}

    version {{ version }}

book.json은 전역변수의 개념 set은 문서 내에서 지역변수의 개념입니다.

    {% set 변수 = "값" %}

    이 변수는 선언한 다음부터 사용 가능합니다.
