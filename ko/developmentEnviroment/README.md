#   개발환경
***
    * 필수개발환경
        * Node.js v4.0.0이상
        * Windows, Linux, Unix, Mac OS X
        
***
    * 설치 명령어
        * npm install gitbook-cli
    * Gitbook 폴더 생성
        * gitbook init [directory]
    * GitBook Build
        * gitbook build
    * GitBook start
        * gitbook serve
    * GitBook plugin install
        * gitbook install

***
    * 사용 버전 및 툴
        * gitbook 3.2.2
        * gitbook-cli 3.2.3
        * git
        * gitBook
        * intellij(markDown 지원)
