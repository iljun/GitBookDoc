# 반복문

반복문 또한 프로그래밍 언어와 사용이 비슷합니다.

{% raw %}
    {% for foo in bar %}
    
    {% endfor %}
    
{% endraw %}

위와 같은 형태로 가능합니다.

book.json에 배열 형태로 선언되어야 사용 가능
```
    {
        "variables" : {
            "v1" : [
                { "name" : "v1"},
                { "name" : "v2"}
            ]
        }
    }
```
